# **App Name**: 汉字练字帖

## Core Features:

- 姓名输入: 用户输入汉字姓名，支持任意字数。
- 练字帖参数设置: 用户可设置每行格子数、描字行数、描字颜色、格子类型（田字格、米字格等）、是否显示拼音、A4 纸边距（固定为2-2）。
- 笔画顺序: 展示汉字的笔画顺序（通过生成图片或矢量动画实现，不依赖数据库）。
- 汉字结构和部首分析: 通过互联网获取汉字信息,分析汉字的结构（如：上下结构、左右结构等）和部首，并显示在练字帖上。
- 练字帖生成: 根据用户输入的姓名和设置的参数，自动生成练字帖的PDF文件，供用户下载或直接打印。

## Style Guidelines:

- 主色调：淡蓝色 (#E6F7FF)，代表清新和专注。
- 背景色：白色 (#FFFFFF)，简洁干净。
- 强调色：浅红色 (#FFA39E)，用于突出显示描字笔画。
- 正文和标题字体：'PT Sans' (sans-serif) ，现代且易于阅读。
- A4纸张页面边距 2cm。内容居中排布。
